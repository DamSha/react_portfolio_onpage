## Getting Started

```
node@v10.16.0 or higher
npm@6.9.0 or higher
git@2.17.1 or higher
```

## How To Use
1. Install dependencies

```bash
$ yarn
```

2. Edit `portfolio.js`

3. Start development server

```bash
$ yarn dev
```

## Linking Portfolio to Github

```javascript
  //  portfolio.js
  githubUserName: 'YOUR GITHUB USERNAME HERE',
```
